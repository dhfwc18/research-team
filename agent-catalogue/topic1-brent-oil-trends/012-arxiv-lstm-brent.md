# arXiv 2409.12376 -- Brent Crude Price Prediction Based on LSTM Model

**Authors:** Yuwen Zhao, Baojun Hu, Sizhe Wang
**Date:** September 2024
**Context:** Low-carbon energy transition

## Description

Deep learning paper applying a three-layer LSTM model to forecast European Brent crude oil
spot prices using EIA spot price data. The model tracks overall price movements well but shows
deviation during sharp price fluctuation periods. Framed in the context of low-carbon energy
transition, targeting policymakers and investors facing oil price uncertainty.
Code and data availability not stated on the abstract page.

## Usefulness Assessment

**Partially relevant** -- Provides a reference LSTM architecture for Brent price forecasting
that could be used as a price-scenario generation component. The low-carbon transition framing
is directly aligned with the project objective. However, this is a forecasting tool, not a
structural model of the oil-renewables relationship.

## Possible Application

- Reference architecture for oil price scenario generation in stress-testing
- Benchmarking forecasting performance against simpler models
- Context for machine learning approaches to energy price modelling
