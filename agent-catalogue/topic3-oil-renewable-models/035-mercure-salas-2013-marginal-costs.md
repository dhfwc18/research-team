# Mercure and Salas (2013) -- Global Economic Potentials and Marginal Costs of Non-Renewable Resources

**Authors:** Jean-Francois Mercure; Pablo Salas
**Journal:** Energy Policy, Volume 63, 2013
**arXiv:** 1209.0708, submitted September 4, 2012

## Description

Paper developing a model of marginal production cost evolution for non-renewable energy
resources over time, integrating resource depletion and consumption patterns. Demonstrates
that marginal costs can be used to simulate long-term average price formation of energy
commodities. Implemented within the FTT:Power global power sector model. Mathematical
framework uses global cost distributions of energy resources enabling bidirectional
calculations between prices and production costs.

## Key Findings

- Marginal production cost trajectories can simulate long-term commodity price formation
- Framework integrates depletion dynamics with demand-side consumption patterns
- FTT:Power implementation provides applied context for the theoretical framework

## Usefulness Assessment

**Relevant** -- Directly applicable to the project's microeconomics model. The cost
distribution approach provides a rigorous method for linking Brent oil prices to marginal
production costs and technology competition with renewables.

## Possible Application

- Marginal cost framework for the Python microeconomics model
- Resource depletion dynamics for long-run oil price scenario construction
- FTT:Power model reference for technology competition between oil and renewables
