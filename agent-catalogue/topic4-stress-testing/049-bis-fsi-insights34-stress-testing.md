# BIS FSI -- Stress-Testing Banks for Climate Change: A Comparison of Practices (FSI Insights 34)

**Organisation:** Bank for International Settlements, Financial Stability Institute
**Date:** July 2021

## Description

Cross-jurisdictional comparison of climate stress-testing practices across supervisors and
central banks. PDF is 613 KB; binary content on fetch. Reviews methodological approaches
(top-down vs bottom-up, physical vs transition risk, scenario sourcing) across multiple
regulatory regimes. Identifies convergence and divergence in practices. Key contribution:
synthesises state of the art across jurisdictions as of mid-2021.

## Usefulness Assessment

**Relevant** -- Provides comparative overview of stress-testing methodologies useful for
selecting the most appropriate approach for the project. The cross-jurisdictional perspective
identifies methodological best practices.

## Possible Application

- Methodology selection guidance for the project's stress-test design
- Reference for justifying scenario choices in the report
- Best practice benchmark for stress-test documentation
