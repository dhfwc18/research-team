# NGFS -- Climate Scenarios Technical Documentation Phase IV (V4.2)

**Organisation:** NGFS / Potsdam Institute for Climate Impact Research (PIK) / IIASA /
University of Maryland / National Institute of Economic and Social Research (NIESR)
**Date:** November 2023 (Phase IV, V4.2)

## Description

Detailed technical documentation for NGFS Phase IV scenarios. Page returned limited content
on fetch (NGFS site); companion PDF is 9.97 MB. Covers the models underlying the NGFS
scenarios (REMIND-MAgPIE for energy-economy, GCAM, MESSAGE-GLOBIOM integrated assessment
models; NiGEM for macroeconomic impacts). Includes technical assumptions for each scenario:
carbon price trajectories, energy price paths, technology cost assumptions, GDP impacts.

## Usefulness Assessment

**Relevant** -- The technical documentation is essential for understanding the model
architecture, parameter assumptions, and calibration method behind the NGFS scenarios.
Required reading for correctly implementing the NGFS framework in the project's stress tests.

## Possible Application

- Technical assumptions (carbon prices, oil price paths, renewable deployment) by scenario
- Model architecture reference for the Backend Engineer's implementation
- Calibration parameters for matching the project's model to NGFS scenario outputs
