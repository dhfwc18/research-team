# IEA -- Global Energy and Climate Model: Scenario Documentation

**Organisation:** International Energy Agency (IEA)
**Date:** November 12, 2025 (continuously updated)

## Description

Documentation for the IEA's principal modelling tool used to generate scenarios across
all IEA publications (WEO, ETP, Renewables, etc.). Six scenarios: Current Policies Scenario
(CPS), Stated Policies Scenario (STEPS), Announced Pledges Scenario (APS), Net Zero by 2050
(NZE), ACCESS scenario, and other analytical frameworks. Integrated hybrid modelling combining
sector-by-sector and region-by-region analysis across fossil fuels, renewables, electricity,
transport, industry, and buildings. Free dataset available: GEC Model Key Input Data
(macro drivers and techno-economic inputs). Licence: CC BY 4.0.

## Scenarios and Key Features

- STEPS: Stated policies baseline
- APS: Full announced pledges implementation
- NZE: Net zero by 2050 pathway
- Each includes macro drivers and techno-economic input data

## Usefulness Assessment

**Relevant** -- The IEA GEC model is the reference model for global energy transition
scenarios. The free key input dataset provides directly usable macro and techno-economic
parameters for calibrating the project's stress-test scenarios.

## Possible Application
