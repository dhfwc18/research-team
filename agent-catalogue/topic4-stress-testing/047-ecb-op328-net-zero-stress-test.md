# ECB -- The Road to Paris: Stress Testing the Transition Towards Net-Zero (OP 328)

**Organisation:** European Central Bank
**Date:** September 2023

## Description

ECB Occasional Paper examining how to stress-test banks and the financial system for
transition risks on the net-zero pathway. PDF is 1.6 MB; binary content partially extractable
on fetch. The paper develops a framework for assessing financial institution exposure to
transition risks (stranded fossil fuel assets, credit quality deterioration in carbon-
intensive sectors) under different transition speed scenarios. Key contribution: integrating
macro-financial and sector-level transmission mechanisms.

## Usefulness Assessment

**Relevant** -- Provides a rigorous stress-testing methodology for transition risks that
is directly applicable to the project. The macro-financial transmission framework is the
state of the art for climate transition stress testing.

## Possible Application

- Stress-testing methodology for the project's macroeconomic scenario stress tests
- Transmission mechanism: Brent price shock -> macro impact -> renewable investment
- Reference framework for the stress-test design in the report
